웹사이트나 웹 애플리케이션 또는 모바일 솔루션의 프로세스와 관련된 서버 측(Server-side)과 데이터베이스를 관리해 주는 테크놀로지

소프트웨어 개발 프로세스에서 서버 측 개발 분야 데이터를 저장하고 관리하며, 웹사이트의 클라이언트 측(Client-side)에서 모든 것이 매끄럽게 작동할 수 있게 해줍니다. 백엔드에서는 프론트엔드에 있는 사용자들이 취하는 행동들을 처리

(1) 시스템 컴포넌트 작업
(2) API 작성
(3) 데이터베이스 통합
